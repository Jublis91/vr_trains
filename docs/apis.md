# API-muistiinpanot

## REST API ja testidata
- JSON Placeholder: https://jsonplaceholder.typicode.com/
- Hyvä testata GET/POST -kutsut ilman tunnuksia.

## Digitraffic Rail API (VR-junat)
- Dokumentaatio: https://rata.digitraffic.fi/swagger/
- Esimerkki live-junat:
  - `GET https://rata.digitraffic.fi/api/v1/live-trains?station=HKI`
- Junasijainnit:
  - `GET https://rata.digitraffic.fi/api/v1/train-locations/latest/{trainNumber}`

## Tehtävä-API ja keskusteluryhmä
- Rautatieliikenne (Digitraffic): https://rata.digitraffic.fi/
- Keskusteluryhmä: https://rata.digitraffic.fi/ (ks. Discussions / yhteisölinkit)
